# HRE4M motion graphics

This folder contains editable Remotion source compositions. The student-facing Astro/Svelte app does not ship React or the Remotion runtime; it serves the rendered, accessible media files from `public/media/`.

## Commands

- `npm run motion:studio` opens the Remotion Studio for visual editing.
- `npm run motion:poster` renders the representative still used as the native-video poster.
- `npm run motion:render` renders the production MP4 used by the Cana lesson.

The first composition, `CanaSign`, is a silent 9-second motion graphic for John 2:1–11. Keep the MP4 silent unless a complete transcript and captions workflow is added to the lesson media record.
